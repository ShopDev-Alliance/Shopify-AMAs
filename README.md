## Conventions

### Folders

- Use the format `YYYY-MM` for the folder name.

### Files

- Use the format `YYYY-MM-DD-AMA-TITLE.md` for the file name.
- Use the following template for the file content:

```markdown
# AMA Title

Date: YYYY-MM-DD

Recording: [AMA Title](URL)

## Hosts

- [Host 1](URL)
- [Host 2](URL)
- [Host 3](URL)

## Notes

```
